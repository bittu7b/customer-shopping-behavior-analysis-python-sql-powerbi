# customer-shopping-behavior-analysis-python-sql-powerbi

## Project Overview
This project analyzes customer shopping behavior using transactional data from 3,900 purchases across multiple product categories. The objective is to uncover insights into customer spending patterns, product preferences, and subscription behavior to support data-driven business decisions.

## Tools & Technologies
- Python (Pandas, NumPy)
- SQL (MySQL)
- Power BI
- Data Visualization

## Project Workflow

### 1. Data Cleaning and Preparation (Python)
- Loaded dataset using pandas
- Checked structure using df.info()
- Generated summary statistics using df.describe()
- Handled missing values in review rating column
- Standardized column names using snake_case

### 2. Feature Engineering
- Created age_group column by binning customer ages
- Created purchase_frequency_days column from purchase frequency data
- Checked redundancy between discount_applied and promo_code_used

### 3. Data Analysis using SQL
SQL was used to answer key business questions such as:
- Revenue generated by male vs female customers
- Customers who used discounts but spent above average
- Top products with highest review ratings
- Comparison of purchase amount by shipping type
- Spending behavior of subscribed vs non-subscribed customers
- Top purchased products within each category

### 4. Data Visualization using Power BI
An interactive dashboard was built to visualize customer behavior and sales insights.

Dashboard highlights:
- Revenue by category
- Revenue by age group
- Customer subscription analysis
- Sales performance by product category

## Business Recommendations
- Promote exclusive benefits to increase subscriptions
- Introduce loyalty programs for repeat buyers
- Optimize discount strategies to balance revenue and margins
- Highlight top-rated and best-selling products in marketing campaigns
- Target marketing efforts on high-value customer segments
